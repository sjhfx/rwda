# rwda
rwda是一个微博数据分析的R包。

## 安装

```R
# install.packages("devtools")
devtools::install_github("sjhfx/rwda")
```
## 使用示例

```R
library(rwda)
df_comments = get_comments(access_token, weiboid)
weiboda(df_comments)
```
